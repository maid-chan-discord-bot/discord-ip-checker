# discord-ip-checker
Just simple discord ip checker<br>

## Install (Windows) <br>
``` pip install -r requirements.txt```

## Install (Linux and mac) <br>
``` pip3 install -r requirements.txt``` 



# Run

## Run (Windows) <br>
``` python main.py ``` Or ``` py main.py ```

## Run (Linux and mac) <br>
``` python3 main.py```
